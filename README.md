# Đại Phú Shopping Cart – Business Analysis Case Study

## Project Overview

This repository presents a Business Analysis case study for the
Shopping Cart module of the Đại Phú silk flower e-commerce website.

The project was completed during a professional internship at Platihub.

## Business Problem

Before a customer proceeds to checkout, the customer needs a clear
and reliable way to:

- Review selected products
- Update product quantities
- Remove unwanted products
- Enter a personalized order note
- Verify item totals and cart totals

The Shopping Cart module was analyzed and designed to support
these needs.

## Key Functions

- View shopping cart products
- Increase and decrease quantity
- Prevent quantity from falling below one
- Calculate item total and subtotal
- Enter an order note of up to 256 characters
- Remove one product
- Remove all products with confirmation
- Display an empty-cart state
- Continue to checkout information
